Documentation Title: Achievement 2 Project - myFlix This project is an API that will be used to return information on various movies to the user. Further more users will be able to register/dregister and add/remove movies with a "favorites"-list.
Request URL Request Parameter HTTP Method Response body data format Request body data format
Returns a list of all movies "/movies" none GET A JSON object holding data about all the movies
Example: {Title: "Watchmen", Description: "", Director: { Name: "Zack Snyder", CV: "" }, Genre: { Name: "Comic-Adaptation", Description:""} }
Returns data about a single movie by title "/movies/:title" :title GET A JSON object holding data about a single movie and in depth data about its genre and director.
Example: {Title: "Watchmen", Genre: {Name:"Drama", Description:""}, Director {Name:"Zack Snyder", Bio:: "", Birth-Year:""}, Actors: [""], Description: ""}
Returns data about a genre by name "/movies/genres/:genreName" :genreName GET A JSON object holding data about a genre
Example: Genre: { Name: "Drama", Description:""}
Returns data about a director by name "/movies/directors/:directorName" :directorName GET A JSON object holding data about a director
Exmple: Director: { Name: "Edgar Wright", CV: "" },
Allows new users to register "/login" none POST A JSON object holding data about new user that was added, with an added _id
A JSON object holding data about the user to add (Username, Password and Email are required)
Example: { Username: "Siegfried", Password:"Roy", Email: "dragonkiller69@gmail.com, FavoriteMovies:"favoriteMovie_Id"}
Allows registered users to login "/login?Username=:Username&Password=:Password" :Username, :Password POST A JSON object holding data about new user that was added, with an added _id and an added bearer-token
The Username and password - parameters
Updates user info "/users/:Username/" :Username PUT A JSON object with the updated user-info A JSON object containing additional information such as { Username: "Siegfried", Password:"Balmung", Email: "dragonkiller69@gmail.com, FavoriteMovies:"favoriteMovie_Id"}
Adds a movie to the user's list of favorites "/users/:Username/:title" :title POST A text message that states a a favorite movie was added
Removes a movie from a user's list of favorites "/users/:Username/movies/:MovieID" :Username, :MovieID DELETE A text message that states a a favorite movie was removed
Deregisters a user "/users/:Username" :Username DELETE A text message that states the user id has been deleted

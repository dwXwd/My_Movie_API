Documentation Title: Achievement 2 Project - myFlix This project is an API that will be used to return information on various movies to the user. Further more users will be able to register/dregister and add/remove movies with a "favorites"-list.
Request URL Request Parameter HTTP Method Response body data format Request body data format
Returns a list of all movies "/movies" none GET A JSON object holding data about all the movies
Example: {Title: "Watchmen", Description: "", Director: { Name: "Zack Snyder", CV: "" }, Genre: { Name: "Comic-Adaptation", Description:""} }
Returns data about a single movie by title "/movies/:title" :title GET A JSON object holding data about a single movie
Example: {Title: "Watchmen", Genre: "", Description: ""}
Returns data about a genre by name "/movies/genres/:genreName" :genreName GET A JSON object holding data about a genre
Example: Genre: { Name: "Drama", Description:""}
Returns data about a director by name "/movies/directors/:directorName" :directorName GET A JSON object holding data about a director
Exmple: Director: { Name: "Edgar Wright", CV: "" },
Returns data about an actor by name "/movies/actors/:actorName" :actorName GET A JSON object holding data about a director
Exmple: Actor: { Name: "Nick Frost", CV: "" },
Allows new users to register "/users"/ none POST A JSON object holding data about new user that was added
Example: { id: 3, name:"Mennis", favoriteMovies:[] }, 		A JSON object holding data about the user to add
Example: { id:, name : "" } }
Updates user info "/users/:id" :id PUT A JSON object with the updated user-info
Adds a movie to the user's list of favorites "/users/:id/:title" :title POST A text message that states a a favorite movie was added
Removes a movie from a user's list of favorites "/users/:id/:title" :title DELETE A text message that states a a favorite movie was removed
Deregisters a user "/users/:id" :id DELETE A text message that states the user id has been deleted


Entity Attributes
Movies ID, Title, Description, Genre, Director, URL to image, Featured (Y/N)
Director ID, Name, Bio, Birth Year, Death Year
Genre ID, Name, Description
Users ID, Username, Password, Email, Birthday, Favorite Movies
